# core-finance-lib
# Estrutura do Data Lake - Liga Financeira

## Camadas de Dados:
- **01_Bronze (Raw):** Dados brutos extraídos via APIs (Yahoo Finance/CVM).
- **02_Silver (Cleaned):** Dados limpos, ajustados por dividendos e padronizados.
- **03_Gold (Refined):** Tabelas prontas para modelos de Asset e Research.

## Tecnologias:
- Linguagem: Python
- Armazenamento: Google Drive (Híbrido)
